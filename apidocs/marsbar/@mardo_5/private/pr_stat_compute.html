pr_stat_compute

PURPOSE

private function to compute statistics for SPM2 design

function [con,stat,Ps,Pc] = pr_stat_compute(SPM,Ic)

DESCRIPTION 

 private function to compute statistics for SPM2 design
 FORMAT [con stat Ps Pc] = pr_stat_compute(SPM,Ic)
 
 Input
 SPM       - SPM design structure

 Output

 stat      - statistic value
