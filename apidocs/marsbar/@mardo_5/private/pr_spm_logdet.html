pr_spm_logdet

PURPOSE

returns the log of the determinant of positive semi-definite matrix C

SYNOPSIS

function [H] = pr_spm_logdet(C)

DESCRIPTION 

 returns the log of the determinant of positive semi-definite matrix C
 FORMAT [H] = pr_spm_logdet(C)
 H = log(det(C))
