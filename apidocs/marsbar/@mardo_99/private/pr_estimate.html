pr_estimate

PURPOSE

Estimation of a General Linear Model

SYNOPSIS

function SPM = pr_estimate(SPM, marsY)

DESCRIPTION 

 Estimation of a General Linear Model
 FORMAT SPM = pr_estimate(SPM, marsY)
 Inputs 
 SPM      - SPM design structure

 Outputs
