Support
~~~~~~~

We hope it is clear how to use MarsBaR, but please let us know if you
have problems. In particular, we would be very glad to hear of any
bugs or inconsistencies.

There is a Discourse forum for MarsBaR; you can search for previous questions
and answers, and see instructions for joining posting on the `MarsBaR
users Discourse site`_ page.
